API接口

API

当前网关提供两种API接口

  1. 基于HTTP的Restful API可以用任何http客户端调用,缺点为只能内网访问,或借助VPN
  2. 基于MQTT的接口,需要程序适配,只要设备接入MQTT服务,即可远程控制,只支持通用MQTT
